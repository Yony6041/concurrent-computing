Para ejecutar el programa se necesita instalar maven, una vez hecho esto vaya al directorio Chinches e instale:

mvn package

Y posteriormente ejecutelo con:

java -cp target/chinches-1.0-SNAPSHOT.jar kas.concurrente.Main
